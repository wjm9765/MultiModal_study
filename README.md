# MultiModal_study

## 멀티 모달 스터디 실습 저장소
이 저장소는 멀티모달 스터디에서 진행 실습 코드를 공유하는 공간입니다.

## 스터디 진행 방식
1. 논문 리뷰: 매주 정해진 멀티모달 관련 논문을 읽고 핵심 내용을 학습합니다.
2. 주제 토의: 논문 내용을 바탕으로 구현해 볼 실습 주제를 함께 토의하고 결정합니다.
3. 결과 공유: 다음 주차 스터디에서 각자 진행한 이전 주차의 실습 결과를 발표하고 공유합니다.

## 주차별 논문 및 실습 내용
- 1주차: CLIP,BLIP
- 2주차: ALIGN - noisy 데이터셋과, 깨끗한 데이터 셋의 모델 성능 비교
- 3주차: AudioCLIP, Whisper - AudioCLIP의 다중 쿼리 간에 모호한 맥락 이해 테스트
- 4주차: LLaVA, MiniGPT-4 - LLava의 전처리 과정 및 선형 레이어 학습 과정 구현
- 5주차: DALL-E 2, Stable Diffusion - Unet의 전체적인 작동 방식 코드 구현
- 6주차: Kosmos-2, Audio Flamingo, Flamingo-2
